What is React Hooks?

Ans: Hooks are a new feature addition in React version 16.8 which allow you to use React feature without having to write a class.

Example: React state and other feature you can use using react hooks

Keep in mind that hooks donot works inside classes

Why Hooks?

  1. There is no confusion with the 'this' keyword, since we are not using class component in react hooks
  2. Hooks helps us to share (reuse) statefull logic without changing component higherkey
  3. Code repeation in the following in class component -
      - Data fetching: Related code will split into componentDidMount and componentDidUpdate
      - Event listeners: in componentDidMount and componentWillUnmount
    Completely unrelated code for Data-fetching & Event-listeners will endup in the same code block componentDidMount
    Because of statefull logic - it is not possible to break down the code into different componet, so react hooks came into picture - it will wrape up the related code into same block, - (useEffect)
