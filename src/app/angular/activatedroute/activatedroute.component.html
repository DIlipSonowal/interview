What is ActivatedRoute?

An interface that Provides access to information about a route, parameters associated with a component that is loaded in an outlet. Use to traverse the RouterState tree and extract information from nodes.

    snapshot: ActivatedRouteSnapshot

    fragment: Observable< string >

    outlet: string
    component: Type< any > | string | null
    routeConfig: Route | null
    root: ActivatedRoute
    parent: ActivatedRoute | null
    firstChild: ActivatedRoute | null
    children: ActivatedRoute[]
